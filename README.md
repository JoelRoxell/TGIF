# TGIF (alpha)
Fast, unopinionated and minimal skeleton project. Start of your projects as quickly as possible and with no hardcore configuration.


## Core Technologies 
* React
* Redux
* React-router
* Webpack
* PostCSS


## Docs 
* NAN

## Quick Start

Install dependencies:

```
npm install
```

Run webpack (HMR) and start develop:

```
npm run dev
```
